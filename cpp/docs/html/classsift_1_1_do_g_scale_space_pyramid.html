Detailed Description

The Difference of Gaussian (DoG) Scale Space Pyramid described in [Lowe 2004] in Section 3.

Creates a DoG scale-space pyramid as described by [Lowe 2004].

Parameters
[in]imageThe original image to create the pyramid from.
[in]octavesThe number of octaves to use in the pyramid. If -1, computes the number of octaves automatically from the size of the image.
[in]stddevThe initial std dev of the Gaussian at each octave.
